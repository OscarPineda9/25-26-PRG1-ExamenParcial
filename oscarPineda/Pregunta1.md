# Respuesta Pregunta 1

## ¿Hay Error en el fragmento ?

final String VENTANA_CERRADA = ":[ ]:";
final String LUZ_APAGADA = ":[º]:";
final String LUZ_ENCENDIDA = ":[*]:";


## Respuesta
No hay error en el fragmento ya que no hay mala practica en la escritura de las variables. Ya que en las variables final deben estar en Mayusculas todas las palabras y si son describidas por dos palabras debe de haber una barrabaja entre esas dos palabras ademas que las variables estan bien explicadas con sus nombres y valores no se mal entienden.
